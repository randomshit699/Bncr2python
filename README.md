# Bncr2python 

无界（Bncr）的python插件兼容层 
## 1.安装 
### 1.1 容器内安装python 
```shell
docker exec -it bncr sh  #进入无界容器内
apk update && apk upgrade  #更新包管理器
apk add python3=3.11.14-r0 py3-pip  #安装python和pip
exit  #退出容器
```
